# spm-bump

> bump version, create tag and publish to npm

-----

## Install

Install spm-bump with npm:

```
$ npm install ibump -g
```

## Usage

```
Usage: ibump [options]

Options:
  -h, --help     output usage information
  -V, --version  output the version number
  -v, --verbose  
  -M, --major    major +1
  -m, --minor    minor +1
  -p, --patch    patch +1
```
